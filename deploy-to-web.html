GitHub + Vercel

Local Server (Development)

# Using Python python -m http.server 8000 # Using Node.js npx serve . # Using Live Server (VS Code extension) # Right-click redesigned-lobby.html > "Open with Live Server"

🎨 Figma Import Guide

Method 1: Direct Import

1. Deploy to web (use any method above) 2. Install "HTML to Figma" plugin 3. Paste your deployment URL 4. Import with all styles preserved

Method 2: Design Tokens

1. Open design/design-tokens.json 2. Use "Figma Tokens" plugin 3. Import all color/spacing/typography tokens 4. Auto-generate design system

Method 3: Screenshot Import

1. Open redesigned-lobby.html in browser 2. Switch to mobile view (393×852) 3. Screenshot each page (#home, #ranking, #profile, #info, #buycoins) 4. Import screenshots to Figma as reference
